「2018台北金鵰微電影展」出爐，中信金一舉拿下五大獎！中信金控昨日指出，中信拍攝的企業形象影片─「人生界外球」榮獲頒發「最佳影片」大獎、「社會共融類」優等及「拍攝技巧類」優等，而「Home Run Taiwan愛就跑！9天8夜不斷電環台馬拉松」全紀錄影片則獲頒「拍攝技巧類」最佳及「社會共融類」優等，兩部影片共獲得5座獎項肯定。中信金控指出，「2018台北金鵰微電影展」是由財團法人台灣永續能源研究基金會及台灣企業永續學院共同主辦，其目的是鼓勵企業透過微電影的力量，以動態形式呈現企業致力於永續發展目標，透過新媒體向社會大眾宣揚永續意識，主辦單位特別邀請資深電影人、名監製與導演李崗出席頒獎，中信金控行政長高人傑代表受獎。高人傑在頒獎典禮上表示，「人生界外球」及「Home Run Taiwan」兩支獲獎影片，皆為中國信託在2016年適逢成立50周年時，為感念臺灣這塊土地的滋養哺育及社會各界的信賴與支持，藉由影片傳遞臺灣良善價值，並為這塊土地無私奉獻的無名英雄致上最高敬意。中國信託企業形象影片「人生界外球」是以臺東縣豐田國小少棒隊築夢國際舞臺為主軸拍攝的企業形象影片，敘述一群資源匱乏的孩子在許多沒有血緣關係卻視如己出的「關鍵人」協助下，一路闖進決賽殿堂的故事，呼應中國信託金控所強調「家」的價值，獲得評審團高度評價，被認為故事精彩動人且充分表達社會議題，拍攝手法透過角色切換交錯及剪輯節奏，讓影片情緒堆砌有力，榮獲「最佳影片」大獎及「社會共融類」、「拍攝技巧類」優等。而「Home Run Taiwan」紀錄片藉由記錄「Home Run Taiwan愛就跑！9天8夜不斷電環台馬拉松」活動，融入多項社會關懷議題，搭配樸實的旁白，讓影片親切且有溫度，影片採用蒙太奇手法節奏佳，整體結構扎實，獲頒「拍攝技巧類」最佳及「社會共融類」優等的肯定。(工商時報)
