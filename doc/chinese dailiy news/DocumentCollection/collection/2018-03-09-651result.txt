沒有人想搬家的老舊公寓，枝葉茂盛幾可相連成「綠色隧道」的行樹，台北市民生社區富錦街已被一家家設計工作室、文創小店、咖啡館，以及各種以「類電影場景」風格裝潢陳設的美食餐廳串連成台北的「文青一條街」與「文創聚落」，其中的〈Cafe Mode木馬咖啡〉別有風情況味，原因不只是它就位在富錦3號公園的邊上，同時餐廳前有庭院、中有天井，得以讓客人享用難得的靜謐與閒情，更耐人尋味的是，菜單上的跨界美食平價好口味，店裝與菜餚都流洩著一股「小清新」的氣質。〈Cafe Mode 木馬〉的老闆林哲毅小時候即赴紐西蘭當小留學生，大學在紐西蘭讀的是飯店管理，學校業畢業後於2000年返台並進到台北君悅酒店餐飲部工作，並從基層逐步高升並先後擔任義大利與中餐廳的經理。學的是餐飲、又在飯店作了10年，林哲毅選擇離開飯店創業。林哲毅生肖屬馬，英文名字Troy又是〈木馬屠城記〉中「特洛伊」，加上他在紐西蘭生活時就愛騎馬。基於和馬的多種緣份，因此決定以木馬為店名。民生社區富錦街的老公寓不少，但若是有庭院又位在公園邊上就不多了，因為生活環境好，當地居民「不喜歡搬家」，故若想找到位在富錦街一樓、前有庭院，旁邊又緊臨公園的店面，那還真難，但林哲毅的運氣就是好，不僅找到了好店面，還遇到了好房東。〈Cafe Mode 木馬〉的設計裝潢都是林哲毅自己設計規畫，品味不俗的他以「馬」為元素凸顯餐廳的意象與視覺，他平常就收集了很多馬蹄鐵、馬造型的公仔、布偶、大大小絲的木馬，如今全部都用在餐廳裡各個不同角落，因此餐廳還多了幾分童趣。〈Cafe Mode 木馬〉的空間很有層次，不同區域各有氛圍與機能，庭院戶外區到了節慶時可以烤肉，室內吧檯可以喝飲料咖啡，靠窗一側可直接擁覽公園綠意，天井區則布置成一獨立小花園，成了親子客人最愛的區域。事實上，富錦街上的咖啡館或餐廳雖多，但有些多元空間的卻少。飯店經理人出來創業，總希望能做出點與眾不同的名堂，因此也比一般人創業會有更多的理想與概念。而林哲毅的創意想法除了用在餐廳的空間，也體現在菜式的形色味上。〈Cafe Mode 木馬〉的食物菜餚看似簡單，卻內蘊著林哲毅的企圖傳遞給客人的非凡體驗。例如，這裡的果汁一律是鮮果現榨，完全不汁也不加糖。麵包則是用天然酵母然發酵，沒有用到人工添加物。而洋蔥、花椰、香茹或鮮蝦等食材，則是天天到市場採買。〈南瓜湯〉不用罐頭，而是將南瓜蒸熟後打成泥，再加了奶油熬煮，所以質地濃稠且湯內還有南瓜塊。〈Cafe Mode 木馬〉菜單上的菜餚「只求好吃，不計菜系血統」。例如以雞肉作餡的〈帕尼尼〉，雞腿肉是用中式米酒醃過水煮，帕尼尼麵包上塗了豆腐乳後再夾餡，中西合璧的味道很好吃。而〈青醬大蝦釜飯〉則是將日本釜飯、義大利燉飯，以及台灣小菜結合後呈現，米飯中拌香菇並用青醬調味、上面舖著鮮蝦，用日式鐵釜炊熟，開蓋後再將起司條拌入飯中增加口感風味，釜飯上桌時還附上一碟豆腐乳讓客人配飯。這樣的巧思創意，也讓這餐廳更有「味道」，並成了一大賣點。(工商 )
