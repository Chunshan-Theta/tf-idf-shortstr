中國信託金控以台東縣豐田國小少棒隊築夢國際舞台為主軸拍攝的企業形象影片─「人生界外球」，榮獲「2018台北金鵰微電影展」頒發「最佳影片」大獎、「社會共融類」優等及「拍攝技巧類」優等，而「Home Run Taiwan愛就跑！9天8夜不斷電環台馬拉松」全紀錄影片則獲頒「拍攝技巧類」最佳及「社會共融類」優等，2部影片共獲5座獎項肯定。「2018台北金鵰微電影展」是由財團法人台灣永續能源研究基金會及台灣企業永續學院共同主辦，主辦單位特別邀請資深電影人李崗頒獎，中信金控行政長高人傑代表受獎。高人傑表示，「人生界外球」及「Home Run Taiwan」兩支獲獎影片，皆為中國信託前（2016）年適逢成立50周年時，為感念台灣這塊土地的滋養哺育及社會各界的信賴與支持，藉由影片傳遞台灣良善價值，並為這塊土地無私奉獻的無名英雄致上最高敬意。「人生界外球」獲得評審團高度評價，被認為故事精彩動人且充分表達社會議題，拍攝手法讓影片情緒堆砌有力，而「Home Run Taiwan」紀錄片藉由記錄「Home Run Taiwan愛就跑！9天8夜不斷電環台馬拉松」活動，融入多項社會關懷議題，搭配樸實的旁白，讓影片親切且有溫度。「人生界外球」在Youtube及Facebook上線短短1個半月就突破468萬人次點閱，觸動4萬多人次分享。「Home Run Taiwan」紀錄片由拍攝團隊從100多組個案中精選出極具代表性的故事，共耗時1年完成，上線時累計1700萬人次瀏覽、超過12萬人次分享。中國信託相信影片傳達愛的循環理念，能展現最大的社會影響力，為台灣帶來永續的正面能量，並向社會大眾傳達環境永續、土地共存、弱勢關懷等企業永續價值，不僅與聯合國「全球永續發展目標」的永續精神相契合，亦展現台灣社會的良善價值與力量。(中國時報)
