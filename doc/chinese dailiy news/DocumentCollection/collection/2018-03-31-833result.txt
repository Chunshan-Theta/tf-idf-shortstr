在二月底的MWC(全球行動通訊大會)及三月份一波新手機發表浪潮後，上半年計畫推出新手機的廠商大多數都已公開了上半年在智慧型手機產業布局。根據韓媒報導，向來都在下半年推出Galaxy Note系列新機的三星，今年可能選擇提早推出Note 9。三星為什麼選擇這樣做，則與自家新品的表現不佳以及對手來勢洶洶則都有關係。《韓國先驅報》(The Korea Herald)旗下的《The Investor》報導指出，根據他們取得的最新產業鏈報告，三星可能會選擇提早讓Galaxy Note 9亮相，其中一個原因就是三星Galaxy S9系列的市場銷量不如預期。韓媒有此推論的基礎在於，他們發現三星顯示器(Samsung Display)在今年四月份就會開始生產Galaxy Note 9的OLED螢幕，比以往的六月份更早。這訊息暗示下一代的Galaxy Note手機最早將可能提前到七月或者八月發表。對產品供應時間表的推論，三星顯示器方面拒絕評論。針對三星Galaxy Note 9的螢幕，據傳將會上看6.38吋，比Galaxy Note 8的6.32吋更大，而這將是三星高階手機有史以來最大的螢幕。《The Investor》指出，Galaxy Note 9可能不會搭載當前中國智慧型手機廠已率先採用的屏下指紋解鎖技術。而除了三星Galaxy S9系列在市場上的表現可能不如預期之外，報導也指出三星提早推出Note 9的因素之一，也包含必須要趕在蘋果新iPhone亮相之前這一點。相對於前幾年，Galaxy Note 9今年要面對的新iPhone列隊更強大。根據先前分析師預測，蘋果今年將會一口氣推出三款新iPhone，包含6.1吋LCD螢幕的款式，還有5.8吋、6.5吋OLED螢幕的款式。相對於三星Galaxy Note系列沒有非常固定的發表時間，蘋果新iPhone多年來則都維持在秋季發表會更新的節奏。回溯近幾年三星Galaxy Note系列發表時間，2017年Galaxy Note 8在8月23發表(台灣地區9月15日上市)；三星Galaxy Note 7是在2016年8月2日發表(三星跳過Galaxy Note 6的命名，直接從Note 5跳到Note 7)；而三星Galaxy Note 5則是在2015年8月13日發表。從發表時間來看，Galaxy Note 7是近年來發表最早的Note系列手機。不過，之後也在全球引發了廣泛的Note 7電池爆炸事件，重挫三星形象。如果三星今年選擇再度提早發表Galaxy Note 9，其手機的安全性勢必再度受到消費者放大檢視。如何在提前推出手機的前提下，確保手機的安全性與競爭力，對於三星而言，相信仍是不簡單的任務。(中時電子報)<span>文章來源：</span><a href="http://www.theinvestor.co.kr/view.php?ud=20180329000579&amp;ACE_MAIN=2"
