年貨夯品吹起養生風，不少人都來買堅果類的點心，當成過年零嘴，像是核桃、夏威夷豆、葵瓜子、花生等。知名插畫粉專「10 Seconds Class - 10秒鐘教室」近日統整一張集合14種核果的「堅果圖」，以手繪插畫風格將各種堅果的特色畫出來，讓網友看了大讚「很有趣的圖，看完瞬間知識爆棚」。粉專「10 Seconds Class - 10秒鐘教室」常把生活健康知識輔以插圖，讓大家可以一張圖秒懂新知，這次將14種堅果擬人化，並且將各自的特色加強畫出來，讓粉絲不再霧煞煞。「10 Seconds Class - 10秒鐘教室」首先解釋，「堅果是植物的一類果實，通常用來指果皮堅硬，且成熟時果皮不裂開的乾果類。口語上的堅果泛指具硬殼的可食用果仁，其實許多在植物學上並不屬於堅果（如花生、開心果、腰果等），食用的是種子部分。」接著一一畫出各堅果的外形，並且文字解說產地與教大家如何辨別。1.榛果（Hazelnuts）：香味獨特，常與巧克力和咖啡搭配。2.開心果（Pistachios）：其硬殼會自然裂開早期稱為「開口笑」。3.腰果（Cashew）：原產熱帶美洲，常以油炸後直接食用。4.巴西堅果（Brazil Nut）：口味佳、營養豐富，固有「堅果之王」的美稱。5.花生（Peanut）：又稱土豆，是日常生活常見的百搭食品。6.栗子（Chestnut）：其常用於食品，其中更以糖炒栗子最常見。7.松子（Pine Nut）：可入藥、做菜，松子茶為韓國的驅寒聖品。8.扁桃仁（Almonds）：台灣常稱其杏仁果，多用來做甜點、牛軋糖。9.杏仁（Apricot）：杏樹的果仁，具藥用價值，又分為南杏和北杏。10.胡桃（Pecan）：香味濃、皮非常硬，需要用胡桃鉗打開。11.核桃（Walnuts）：形似人腦，常用在蛋糕與冰淇林配料中。12.夏威夷果（Macadamias）：雖然它叫夏威夷果，但其實原產於澳洲。13.葵花籽（Polly Seed）：常見的零嘴（鼠類也愛吃），也會用來榨油。14.南瓜籽（Pumpkin Seed）：南瓜種子，有大量的蛋白質和膳食纖維。超用心的圖文整理，讓許多網友看了紛紛驚呼，「什麼！我常吃的杏仁竟然不是杏仁」、「 以前賣堅果雜糧時，有客人嫌棄夏威夷果、腰果居然不是台灣產的，真的需要補充一下知識」、「來認識堅果！」，還有人開玩笑表示，「很像各種大便啊」、「松子是粉刺」，而「10 Seconds Class - 10秒鐘教室」也幽默回應「你不要降子（這樣子）」。(中時電子報)
